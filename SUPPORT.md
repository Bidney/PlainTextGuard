# Support

For beta support, use GitHub issues:

https://github.com/bidney/PlainTextGuard/issues

When reporting a bug, include:

- Chrome version
- Extension version
- Source site, for example ChatGPT or Claude
- Mode used, for example ASCII Warning or Auto ASCII Copy
- Whether the copy action used selected text or a site-provided copy button
- A short sample text if it is safe to share

Do not include passwords, API keys, client data, personal data, or private content in issue reports.
